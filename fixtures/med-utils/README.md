# med-utils

A small JavaScript utility library for testing swarm-cp orchestration.

## Modules

- `src/strings.js` — string utilities (slugify, truncate, capitalize)
- `src/arrays.js` — array utilities (chunk, unique, groupBy)
- `src/numbers.js` — number utilities (clamp, round, inRange)
- `src/objects.js` — object utilities (pick, omit, isEmpty)
- `src/dates.js` — date utilities (formatISO, addDays, diffDays)
- `src/predicates.js` — predicate utilities (isEmail, isUrl, isUuid)
